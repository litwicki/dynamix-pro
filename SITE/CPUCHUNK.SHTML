A simple system speed checking utility for all Windows systems
RELEASED: APRIL 21, 2001 - 21:31 PST

Program Description:
This program was created to provide system builders and computer technicians with a reliable, streamlined, and overall, very reliable way to test their computer's performance.

Usage Notes:
To obtain optimal performance ratings on your systems, please verify that the following has been done BEFORE running CPU-chunker:

  • Close all programs running on your system except Windows. 
    This includes any instant messaging / system temperature monitoring / music / web browsing programs.
     
  • Disable any internet sign-on services or email checking services.
    This keeps your system from dialing into the internet accidentally during the benchmark.
     
  • Disconnect from the internet.
     
  • Move the program to your C:\ Directory, or another directory that you can easily navigate to from a DOS prompt.
    The program file is named "cpuchunk.exe"
     
  • Get to a Windows DOS Prompt.
    To do this in Windows 95 / 98 / ME: Go to START -> RUN -> Type in the word "command" (without quotes) and hit ENTER.
    To do this in Windows NT / 2000: Go to START -> RUN -> Type in the word "cmd" (without quotes) and hit ENTER.
     
  • Navigate to the program's directory in MS-DOS.
    To get to the C:\ Directory, type in "cd_\"  (use the back slash... NOT the forward slash found under the "?" on the keyboard.)
    Put a space where the underline is in the command above.
     
  • Press "ALT" + "ENTER" to make the DOS window fill your monitor's screen. 
    The start button, system clock, and all other Windows-looking things should be missing from your screen.
    All that should show on your screen is the black of the MS-DOS box and the gray C:\ prompt.

Running the Program:

  • At the DOS prompt, type in "cpuchunk" and hit ENTER.
  • The program will proceed to load up.
  • Read and follow all on-screen instructions.
  • The tests will run.  If an error occurs or you need to abort, press "CTRL" + "BREAK" on your keyboard at any time and the program will end safely.
  • The results will be displayed.
  • The results are also written to a file on your C:\ root directory. - The file name is "CPUCHUNK.TXT"
  • This file can be opened with any ordinary text reader like notepad.
